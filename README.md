# Clone This Project
` cd project-name `

# Install package
` composer install ` ` npm i `

# Generate Application Key
` php artisan key:generate `

# Change file name 
`.env.example` to `.env`

# Start Tailwind Service
` npm run dev `

# Start Laravel Project
` php artisan serve `
